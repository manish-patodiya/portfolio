## Title
Evaidyak – Multi-Clinic Healthcare Platform

## Description
A production-grade healthcare platform enabling clinics to manage patient data, records, and communication efficiently.

## Stack
PHP, CodeIgniter, MySQL, AWS (S3, SNS, SES), Razorpay

## Impact
- Live production system
- Used by multiple clinics
- Handled real users and payments

## Kind
Healthcare SaaS Platform

## Links
https://www.cliniqtec.in/

---

## caseStudyContent
Evaidyak was one of my earliest production-level projects where I worked directly with real clients and live systems.

The platform supports multiple clinics, each with its own database schema, enabling scalable and isolated data management.

---

## architecture
- Multi-tenant database architecture (separate schemas)
- AWS integrations:
  - S3 for storage
  - SNS for notifications
  - SES for emails
- Razorpay payment gateway integration
- MVC architecture using CodeIgniter

---

## learnings
- Handling real production systems
- Debugging live issues
- Working with clients directly
- Designing multi-tenant systems

## challenges
- Managing multiple databases simultaneously
- Ensuring data isolation and security
- Handling production bugs under pressure
- Integrating payment systems reliably
