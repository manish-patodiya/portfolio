## Title
MSG Sports – Tournament Management Platform

## Description
A full-stack platform built to automate internal sports tournaments, replacing manual processes with a digital system.

## Stack
React.js, Node.js, Express.js, MySQL, Tailwind, Material UI

## Impact
- Used by 200+ employees
- Reduced manual work by 80%
- Improved coordination and transparency

## Kind
Full Stack Web Application

## Links
https://github.com/manish-patodiya/msg-sports

---

## caseStudyContent
MSG Sports was built to digitize the management of company sports events. It introduced structured workflows for admins, captains, and players.

---

## architecture
- Role-based access system (Admin, Captain, Player)
- REST API backend
- MySQL database for structured data
- React frontend with dynamic UI

## learnings
